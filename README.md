# HR Workforce & Performance Analytics 👥📊

## Project Overview
This project focuses on analyzing Human Resources data to improve workforce management. By examining employee demographics, performance scores, and termination reasons, the project provides a data-driven approach to understanding employee retention, compensation fairness, and recruitment channel efficiency.

## Project Workflow
The repository follows a structured data analysis process:

1.  **Raw Data (`Data Before Cleaning`):** Contains the original workforce dataset, including personal details, position history, and initial performance notes with varied formatting.
2.  **Processed Data (`Data After Cleaning`):** The standardized dataset where IDs are mapped, dates are formatted, and gender/status fields are cleaned for accurate reporting.
3.  **Core HR Metrics (`Measures`):** Key performance indicators (KPIs) calculated from the data:
    * **Turnover & Retention:** Calculation of turnover rates and analysis of termination reasons (e.g., career change, retirement, performance).
    * **Compensation Analysis:** Salary breakdowns by department (IT, Production, Sales) and position.
    * **Performance Tracking:** Distribution of performance scores (Exceeds, Fully Meets, PIP) across the organization.
    * **Absence Management:** Tracking total and average absences per employee and department.
4.  **Strategic Dashboard (`DashBoard`):** A high-level visual summary of the most critical HR metrics for executive decision-making.

## Key Insights Analyzed
- **Recruitment Source Effectiveness:** Evaluating which platforms (LinkedIn, Indeed, Referrals) provide the highest-performing employees.
- **Demographic Diversity:** Analyzing the workforce composition by gender, marital status, and race.
- **Operational Efficiency:** Measuring the correlation between employee engagement scores and productivity.
- **Departmental Insights:** Comparing average salaries and employee satisfaction levels across different company departments.

## Tools Used
- **Microsoft Excel:** For advanced data cleaning and pivot table analysis.
- **Statistical Modeling:** Used for calculating Turnover Rates and Absence Ratios.

## How to Explore
- Use `Data After Cleaning.csv` for any further data science or visualization tasks.
- Refer to `Measures.csv` to see the numerical logic behind the HR KPIs.
